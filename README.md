# ClickyGame
# Basic overview
The user has to click on boxes in a grid, the score will be based on the accuracy of the clicks and time.
It will be a square box and random boxes will appear anywhere on the grid. Only one box will display at a time, and once you click it a new box spawns.

The game tests your clicking accuracy under stress.

# Properties
A class of grid
A class of box with its own id, property if clicked or to be clicked.
A point based system, giving more points if you clicked faster, and takes away points if missing.
The negative points will also be weighted depending on how far from the target they are.
